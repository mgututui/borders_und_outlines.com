CSS Borders

CSS border – Rahmen

border ist eine Kurzschrift, die alle Eigenschaften eines sichtbaren Rahmens um ein HTML-Element zusammenfasst. Drei Angaben sind erforderlich, damit der Browser einen sichtbaren Rahmen rendert: border-color für die Farbe des Rahmens, border-style für Linien, Punkte oder doppelte Linien und border-width für die Breite von border.
  • Farbe border-color
  • Breite border-width
  • Stil border-style

CSS Border Style

Die Eigenschaft border-style gibt an, welche Art von border angezeigt werden soll.
Folgende Werte sind erlaubt:
  • dotted- Definiert einen gepunkteten Rand
  • dashed - Definiert einen gestrichelten Rand
  • solid - Definiert einen soliden Rand
  • double - Definiert einen doppelten Rand
  • groove - Definiert einen 3D-Rillenrand. Der Effekt hängt vom Randfarbenwert ab
  • ridge - Definiert eine 3D-Kantenlinie. Der Effekt hängt vom Randfarbenwert ab
  • inset - Definiert eine 3D-Einfügungsgrenze. Der Effekt hängt vom Randfarbenwert ab
  • outset - Definiert einen 3D-Anfangsrand. Der Effekt hängt vom Randfarbenwert ab
  • none - Definiert keinen Rand hidden - Definiert einen versteckten Rand
Die border-style-Eigenschaft kann einen bis vier Werte haben (für den oberen Rand, den rechten Rand, den unteren Rand und den linken Rand).

CSS Border Width

Die Eigenschaft border-width gibt die Breite der vier Rahmen an.
Die Breite kann auf eine bestimmte Größe (in px, pt, cm, em usw.) oder mithilfe eines der drei vordefinierten Werte eingestellt werden: thin, medium, oder thick.
Die Eigenschaft border-width kann einen bis vier Werte haben (für den oberen Rand, den rechten Rand, den unteren Rand und den linken Rand).

CSS Border Color

Mit der Eigenschaft border-color wird die Farbe der vier Rahmen festgelegt.

Die Farbe kann eingestellt werden durch:

name - Geben Sie einen Farbnamen an, z. B. "red".
Hex - Geben Sie einen hexadezimalen Wert an, z. B. "# ff0000".
RGB - Geben Sie einen RGB-Wert wie "rgb (255,0,0)" an.
transparent

Die border-color Eigenschaft kann einen bis vier Werte haben (für den oberen Rand, den rechten Rand, den unteren Rand und den linken Rand).

Wenn die border-color nicht festgelegt ist, wird die Farbe des Elements übernommen.
