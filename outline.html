CSS Outline

Ein Umriss ist eine Linie, die um Elemente AUSSERHALB der Ränder gezogen wird, um das Element "hervorzuheben".


CSS hat die folgenden outline Eigenschaften:
  • outline-style
  • outline-color
  • outline-width
  • outline-offset
  • outline

CSS Outline Style

CSS Outline Color

Mit der Eigenschaft outline-color wird die Farbe der Kontur festgelegt.

name - Geben Sie einen Farbnamen an, z. B. "red".
Hex - Geben Sie einen hexadezimalen Wert an, z. B. "# ff0000".
RGB - Geben Sie einen RGB-Wert wie "rgb (255,0,0)" an.
invert - führt eine Farbumkehrung durch (wodurch sichergestellt wird, dass der Umriss unabhängig vom Farbhintergrund sichtbar ist)

CSS Outline Width

Die Eigenschaft outline-width gibt die Breite der Kontur an und kann einen der folgenden Werte annehmen:

thin (typisch 1px)
medium (typisch 3px)
thick (typisch 5px)
Eine bestimmte Größe (in px, pt, cm, em, etc)

A thin outline.

A medium outline.

A thick outline.

A 4px thick outline.

CSS Outline - Shorthand property

Die Eigenschaft outline ist eine Kurzform zum Festlegen der folgenden einzelnen Gliederungseigenschaften:

outline-width
outline-style
outline-color

A dashed outline.

A dotted red outline.

A 5px solid yellow outline.

A thick ridge pink outline.

CSS Outline Offset

Die Eigenschaft outline-offset fügt einen Abstand zwischen einem Umriss und der Kante / dem Rand eines Elements hinzu. Der Abstand zwischen einem Element und seiner Kontur ist transparent.

Das folgende Beispiel gibt einen Umriss von 15 Pixel außerhalb der Randkante an:
